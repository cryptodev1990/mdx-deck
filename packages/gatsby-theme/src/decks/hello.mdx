
# Hello

---

This is the MDX Deck Gatsby theme

---

`npm i @mdx-deck/gatsby-theme`

---

```js
// gatsby-config.js
module.exports = {
  __experimentalThemes: [
    '@mdx-deck/gatsby-theme',
  ]
}
```

---

`mkdir src src/decks`

---

1. Add an MDX deck to `src/decks`
2. Navigate to `http://localhost.com/decks`
