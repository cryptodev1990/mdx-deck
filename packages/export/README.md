# MDX Deck Export CLI

```sh
npm i -D @mdx-deck/export
```

```sh
mdx-deck-export deck.mdx
```

## Options

```
-d --out-dir    Output directory
-f --out-file   Output filename
-p --port       Server port
-w --width      Width in pixels
-h --height     Height in pixels
--no-sandbox    Disable puppeteer sandbox
```
